O try-catch é uma estrutura usada para tratar erros durante a execução de um programa, evitando que ele pare de funcionar de forma inesperada.

Na Experiência do Usuário (UX), ele contribui para a prevenção de erros ao permitir que o sistema lide com falhas de forma controlada, exibindo mensagens claras e mantendo o funcionamento estável. Isso reduz a frustração do usuário e torna o sistema mais confiável
